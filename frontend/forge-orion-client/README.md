# Client Management Project

## Project Overview

This project is a **Client Management System** where you can:

- Add new clients (GST, PAN, phone, email, etc.)
- Edit, view, and delete clients
- Set permissions, billing, and integration settings
- Upload documents (PAN, GST, address proof)
- Preview client summary before saving
